freeCodeCamp Backend API Projects - Timestamp Microservice

Simple API to convert UNIX timestamp to Natural Language Date, and vice versa.
Either are accepted as valid inputs.

Example usage:
https://ms1-ts-micro.glitch.me/api/timestamp/December%2015,%202015
https://ms1-ts-micro.glitch.me/api/timestamp/December 15, 2015
https://ms1-ts-micro.glitch.me/api/timestamp/1450137600

Example output:
{ "unix": 1450137600, "natural": "December 15, 2015" }
