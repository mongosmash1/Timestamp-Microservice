freeCodeCamp Timestamp Microservice

Simple API to convert UNIX timestamp to Natural Language Date, and vice versa.
Either are accepted as valid inputs.

Example usage:
https://mongosmash1-timestamp-microservice.glitch.me/December%2015,%202015
https://mongosmash1-timestamp-microservice.glitch.me/December 15, 2015
https://mongosmash1-timestamp-microservice.glitch.me/450137600

Example output:
{ "unix": 1450137600, "natural": "December 15, 2015" }
